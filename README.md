## 🛠️ Installation & Usage

Get started in seconds.

### 1. Installation

```bash
# Install via NPM
npm install ai-text-humanizer

# Or via Docker
docker run -v $(pwd):/data ghcr.io/your-repo/ai-text-humanizer
```

### 2. Basic Usage

```bash
ai-text-humanizer --input "input.txt" --output "output_humanized.txt"
```

### 3. Command Line Options

```bash
ai-text-humanizer
  --input <file>        # Input text file
  --output <file>       # Output text file
  --mode <normal|advanced> # Processing mode
  --verbose             # Show processing logs
```

### 4. API Usage (Pro Only)

```javascript
const humanizer = require('ai-text-humanizer');

humanizer.process('Your AI-generated text here', 'advanced')
  .then(result => console.log(result.humanizedText));
```
